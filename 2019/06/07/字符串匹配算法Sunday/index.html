字符串匹配算法Sunday

发表于 2019-06-07 | 分类于 算法
一种字符串匹配算法，时间复杂度为O(n)。

Sunday算法在匹配失败时关注的是主串中参加匹配的最末位字符的下一位字符。

  • 如果该字符没有在模式串中出现则直接跳过，即移动位数 = 模式串长度 + 1；
  • 否则，其移动位数 = 模式串长度 - 该字符最右出现的位置(以0开始) = 模式串中该字符最右出现的位置到尾部的距离 + 1。
